# Saraha Frontend

This is the frontend for the Saraha-like anonymous messaging app, built with Next.js and React. The frontend provides a modern, responsive UI for user registration, login, profile management, and sending/receiving anonymous messages.

## Features

-   User authentication and session management
-   Responsive design with Tailwind CSS
-   Profile pages and public sharing links
-   Anonymous message sending interface
-   Dashboard for managing received messages

## Getting Started

1. **Install dependencies:**

    ```sh
    npm install
    ```

2. **Set up environment variables:**

    - Create a `.env.local` file in the frontend root with:
      `     NEXT_PUBLIC_API_URL=http://localhost:5000/api
`

3. **Run the development server:**

    ```sh
    npm run dev
    ```

4. **Build for production:**
    ```sh
    npm run build
    npm start
    ```

## Folder Structure

-   `src/components/` - Reusable UI components (Navbar, forms, etc.)
-   `src/pages/` - Next.js pages (routes)
-   `src/utils/` - Utility functions and helpers

## Usage

-   Register or log in to create your anonymous message link.
-   Share your link to receive anonymous messages.
-   View and manage your messages in the dashboard.
